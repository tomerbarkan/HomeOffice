Fire FX is a set of customizable fire simulation particle systems. 
The particle systems can easily be dragged and dropped to the scene,
then adjusted with the included particle scale tool.

Features

• 65+ Shuriken Particle Systems
• Great starting points for fire and explosion effects
• Includes Scene and preview GUI script
• Easy Drag and Drop Prefabs
• Includes particle scale tool for easy scaling
• Fully compatible with Unity 4 and Unity 4 pro
